# TodoNotes 用户使用帮助

> 最后更新：2026-03-16  
> 适用版本：M0.13-R18（当前仓库）

TodoNotes 的目标是把“混在一页便签里的任务”拆成“可导航的任务页面”。
你可以把任意要点拆成子任务，在子任务页面记录细节，再通过面包屑、书签和历史导航快速切换，减少来回翻找带来的心智负担。

## 你会得到什么

和传统便签相比，TodoNotes 的核心差异是：

- 任务可以无限拆分为子任务
- 每个子任务都有独立页面，不会把细节挤在一处
- 父子任务通过链接卡片、面包屑、书签串起来
- 同一个任务可在多个窗口查看，编辑后自动同步

一句话理解：它是“可钻取的任务便签”，不是“单页堆内容的便签”。

---

## 3 分钟快速上手

1. 在 Library 左上点击`新建`，创建一个根任务。
2. 在右侧详情页写下你的要点（每行一个要点更好）。
3. 将某一行转换为子任务：
   - 选中该行后按`Ctrl/Cmd + Shift + T`，或
   - 右键选择`转换为子任务`
4. 点击生成的子任务链接卡片，进入子任务页面补充细节。
5. 右键任务树里的任务，选择`打开置顶便签`，把当前任务放到桌面侧边持续跟进。
6. 在 Sticky 里用面包屑、书签、待处理快速跳转。

## 两个窗口怎么配合

### Library（任务总览与整理）

适合“组织任务结构”：

- 左侧任务树（层级管理）
- 右侧任务详情（内容编辑）
- 顶部搜索 + 标签页筛选（进行中/已完成/回收站/归档）

### Sticky（执行与记录）

适合“边做边记”：

- 桌面便签窗口，可切换置顶
- 顶部书签栏与待处理列表，便于上下文来回切换
- 内置番茄钟，适合专注时段

说明：同一个顶层任务下，多个 Sticky 会共享书签和皮肤设置。

## 任务管理（Library）

### 新建任务

- 点击`新建`创建根任务
- 在“添加任务至收集箱”输入框输入后回车，可快速落一条根任务

### 标题规则

- 同一父任务下，兄弟子任务不能重名
- 不同父任务下允许同名
- 根任务允许同名

### 完成、归档、回收站

- 勾选任务树 checkbox 可标记完成/未完成
- 右键任务可`归档`或`取消归档`
- 右键父任务可`归档已完成子任务`：批量归档完成子任务并移除父页面中的链接卡片
- 删除使用`删除到回收站`，可在回收站`从回收站恢复`
- 回收站保留 30 天后自动清理

### 拖拽改层级

- 拖到某个任务上：变成其子任务
- 拖到列表空白区：提升为根任务

当前限制：暂不支持同级精确排序，只支持层级变化。

## 编辑与子任务拆分

### 支持的编辑内容

- 段落、标题、无序/有序列表
- 待办事项（task list）
- 引用、代码块
- 图片粘贴（自动写入内容）

### 标题折叠

- 标题左侧的小三角可以折叠该标题下的正文内容
- 折叠范围会一直到下一个同级标题，或更高级标题为止
- 只有“标题后面确实还有内容可收起”时，才会显示这个小三角
- 折叠状态只在当前窗口临时生效；关闭窗口或重开应用后会恢复展开
- 创建标题有两种常用方式：
  - 在行首输入 `# `、`## `、`### ` 后再按空格。这里的 `#` 数量表示标题层级，`#` 是一级标题，`##` 是二级标题，`###` 是三级标题。
  - 使用快捷键：mac 是 `Cmd + Alt + 1~6`，Windows / Linux 是 `Ctrl + Alt + 1~6`。数字 `1~6` 分别对应一级到六级标题。

### 子任务入口

#### 1) 添加子任务（新建）

- 详情页右键菜单`添加子任务`
- Sticky 右键菜单`添加子任务`

#### 2) 转换为子任务（把当前行变成链接卡片）

- 快捷键：`Ctrl/Cmd + Shift + T`
- 右键菜单：`转换为子任务`

规则：仅支持单行转换；未选中文本时，会按光标所在当前行转换。

#### 3) 插入已有子任务链接（恢复入口）

- Library：详情页顶部`插入已有子任务`下拉
- Sticky：右键菜单`插入子任务`（二级展开列表）

说明：候选只显示“当前正文尚未引用”的子任务。

### 链接卡片可做的操作

- 点击卡片：进入子任务页面
- 勾选卡片 checkbox：切换完成状态
- 右键卡片：
  - `打开子任务`
  - `在新便签中打开`
  - `移动到...`（移动到其他父任务）
  - `删除链接块`

说明：删除链接块只删除“入口”，不会删除子任务本身。

### 重命名

- 任务树中双击标题
- Library 详情页顶部双击标题
- Sticky 顶部双击标题

重命名后，链接卡片、书签、关联待办会同步更新。

## 导航与专注（Sticky）

### 面包屑导航

- 点击祖先节点：跳转到对应层级
- 点击当前节点：弹出同级任务列表，快速切换兄弟任务

### 历史导航

- 顶部前进/后退按钮
- 快捷键：`Alt + ←`、`Alt + →`

### 书签

- 右键编辑区：`添加当前页到书签`
- 点击书签名快速跳转
- 点击`×`可移除书签
- 悬停书签会显示层级路径（从顶层下一层到当前页）

### 待处理

- 右键某段文本：`添加文本块到待处理`
- 点击书签栏`待处理 (n)`展开列表
- 点击条目可跨页面跳转并定位到该文本块末尾

### 番茄钟

- 单击：开始 25 分钟专注
- 计时中单击：暂停
- 暂停后单击：继续
- 双击：停止并重置

完成专注后自动进入 5 分钟休息。

### 皮肤与置顶

- 点击`🎨`：打开皮肤面板（颜色 + 透明度）
- 点击`📌/📍`：切换窗口置顶

## 快捷键一览

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl/Cmd + Shift + T` | 将当前行或选中文字转换为子任务 |
| `Ctrl/Cmd + Shift + S` | 切换当前行待办状态（任务列表命令） |
| `Alt + ←` | 历史后退 |
| `Alt + →` | 历史前进 |
| `Tab` | 在待办项中增加缩进层级 |
| `Shift + Tab` | 在待办项中减少缩进层级 |

## 推荐工作流

### 每日计划

1. 在 Library 用“收集箱”快速记下今天要做的事项。
2. 把复杂事项拆成子任务（每个子任务一个页面）。
3. 打开 Sticky，固定在桌面执行。
4. 把临时中断内容加入`待处理`，避免丢上下文。
5. 当天结束前执行一次`归档已完成子任务`，保持页面清爽。

### 长周期项目

1. 根任务写里程碑。
2. 每个里程碑拆成子任务。
3. 子任务页面只记录该子任务的决策、步骤和结果。
4. 用面包屑在“全局-局部”之间来回切换，不在一页里堆信息。

---

## 数据与同步规则

- 数据默认保存在本机（SQLite），不依赖云端服务
- 同一任务在多窗口编辑时，采用“最后一次写入生效”
- 任务更新后，其他窗口会自动刷新显示
- 应用重启后会恢复窗口状态（位置、导航路径、书签、皮肤等）

## 常见问题（FAQ）

### 1. 为什么提示“同一父任务下已存在同名子任务”？

你正在同一个父任务下创建/重命名重复标题。改一个标题即可；不同父任务下允许同名。

### 2. 我删除了父页面里的链接卡片，子任务不见了吗？

没有。你只删除了入口。可通过“插入已有子任务链接”重新加入。

### 3. 为什么在 Sticky 里右键看不到“转换为子任务”？

该菜单只在“当前选区可单行转换”时显示。多行选区或跨节点选区不会出现。

### 4. 多个窗口同时改一个任务会怎样？

以最后保存为准。建议同一时段只在一个窗口集中编辑同一任务，减少覆盖。

### 5. 为什么有时“插入子任务”列表是空的？

候选列表会过滤掉“已在当前正文引用”的子任务；如果都已引用，会显示暂无可选子任务。

### 6. 任务树拖拽后顺序和预期不完全一致？

当前拖拽只支持“改父子层级/提升为根任务”，不支持同级精确插入排序。

### 7. 可以把一个子任务移动到另一个父任务吗？

可以。右键该子任务链接卡片，选择`移动到...`并选择目标父任务。

### 8. 右键菜单会不会被便签窗口裁切？

Sticky 里的右键菜单已改为独立弹窗，不会被便签边界裁切；长菜单支持滚动。
