## 开发顺序建议

1. **阶段 0** → 项目初始化，搭建骨架
2. **阶段 1 + 7** → 数据层 + IPC，确保数据流通
3. **阶段 2** → 窗口系统，确保多窗口可用
4. **阶段 4 + 5** → 编辑器 + 详情页，核心编辑体验
5. **阶段 3** → 库面板，任务管理入口
6. **阶段 6** → 提醒系统
7. **阶段 8 + 9** → UI完善 + 跨平台
8. **阶段 10** → 测试与优化
9. **P1** → 迭代增强
